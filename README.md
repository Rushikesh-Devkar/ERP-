# 🏫 School ERP System

A complete School ERP (Enterprise Resource Planning) System built using Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript.

## 🚀 Project Status

⚠️ Currently Under Development

### Completed Modules

* Login & Authentication
* Dashboard
* Student Management
* Teacher Management
* Staff Management
* Role-Based Access Control

### Upcoming Modules

* Attendance Management
* Fees Management
* Examination System
* Report Generation
* Leave Management
* Notifications
* Timetable Management

## 🛠 Technology Stack

### Backend

## 👥 User Roles

### Admin

* Manage Students
* Manage Teachers
* Manage Staff
* Manage Academic Records
* View Reports

### Teacher

* Manage Attendance
* Upload Marks
* View Student Information

### Student

* View Profile
* View Attendance
* View Marks
* View Fee Status
* Apply for Leave

### Principal

* Monitor School Activities
* Approve Leave Requests
* View Reports & Analytics
